# 🔥 Topic Generator Web应用使用指南

## 🚀 快速开始

### 方法1：一键启动（推荐）

双击运行 **`启动应用.bat`** 文件即可！

应用会自动：
1. ✅ 检查Docker环境
2. ✅ 安装所有依赖
3. ✅ 启动Web界面
4. ✅ 在浏览器中打开

**Docker版本**：使用Docker容器运行，环境隔离，更稳定

---

### 方法2：本地运行（备用）

双击运行 **`启动应用_本地.bat`** 文件

**要求**：
- Python 3.8+
- 已配置VPN代理（白鲸加速器，端口10810）

## 💡 使用技巧

### 1. 定时抓取

建议每天运行1-2次：
- 上午 9:00 - 获取夜间热点
- 下午 6:00 - 获取当日热点

### 2. 数据备份

所有报告自动保存在 `output/` 目录：
- 文件名格式：`hot_topics_YYYYMMDD_HHMMSS.md`
- 可以随时查看历史数据

### 3. 搜索话题

在"历史记录"页面：
- 使用搜索框快速查找特定日期的报告
- 点击"查看详情"查看完整内容

## 🛠️ 故障排除

### 问题1：应用无法启动

**解决方法**：
1. 确保Docker Desktop正在运行
2. 检查白鲸加速器是否开启
3. 查看命令行窗口的错误信息

### 问题2：抓取失败

**可能原因**：
- VPN未连接或端口错误
- YouTube API配额用完（每天重置）
- 网络连接问题

**解决方法**：
1. 检查侧边栏的环境变量状态
2. 确认白鲸加速器正在运行（端口10810）
3. 重启应用再试

### 问题3：历史记录为空

**解决方法**：
1. 确保至少成功运行过一次抓取
2. 检查 `output/` 目录是否存在
3. 查看命令行输出的错误信息

## 🔧 高级配置

### 修改抓取平台

编辑 `config.py` 文件：

```python
'enabled_platforms': [
    'hackernews',
    'youtube_tech_api',
    'youtube_edu_api',
    'twitter_tech',
    'twitter_edu',
    'toutiao',
    'bilibili',
    'baidu',
],
```

注释掉不需要的平台即可。
