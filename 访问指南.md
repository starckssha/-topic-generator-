# 🌍 快速访问指南

## 立即使用（3种方式）

### 方式1️⃣：本地访问（只有你的电脑）

```bash
# 1. 双击启动
启动系统.bat

# 2. 浏览器访问
http://localhost:5000/
```

---

### 方式2️⃣：局域网访问（同一WiFi的设备）

```bash
# 1. 双击启动
局域网访问.bat

# 2. 你的电脑访问
http://localhost:5000/

# 3. 手机/平板访问（同一WiFi）
http://192.168.31.8:5000/
```

**设备列表**：
- 📱 你的手机（浏览器）
- 📱 你的平板
- 💻 家里其他电脑
- 💻 公司其他电脑

### 方式3️⃣：公网访问（全世界都能访问）

#### A. 快速部署到服务器（推荐）

**你需要**：
- WinSCP（下载：https://winscp.net/）
- MobaXterm（下载：https://mobaxterm.mobatek.net/）

**步骤**：

1. **上传项目到服务器**
   ```
   打开WinSCP → 连接到 101.43.15.66
   用户名：root
   密码：sx@123456

   上传整个文件夹到：
   /var/www/topic-generator/
   ```

2. **SSH登录服务器**
   ```
   打开MobaXterm → SSH连接
   主机：101.43.15.66
   用户：root
   密码：sx@123456
   ```

3. **执行部署命令**
   ```bash
   cd /var/www/topic-generator
   chmod +x deploy.sh
   ./deploy.sh
   ```

4. **访问系统**
   ```
   🌍 全球访问地址：
   http://101.43.15.66/
   ```

#### B. 手动部署（如果自动脚本失败）

```bash
# 1. 安装依赖
pip3 install flask flask-cors pymysql requests

# 2. 启动服务
cd /var/www/topic-generator
python3 run_web.py

# 3. 后台运行（推荐）
nohup python3 run_web.py > server.log 2>&1 &
```
